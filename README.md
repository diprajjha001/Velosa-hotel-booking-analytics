# 🏨 Velosa Hotel Booking Analytics Dashboard

An interactive **Power BI dashboard** designed to analyze hotel booking performance, revenue trends, guest behavior, cancellation rates, booking channels, and loyalty program effectiveness for **Velosa Hotels & Resorts**.




---

## 📌 Project Overview

The goal of this project is to provide hospitality stakeholders with a centralized analytics solution to monitor key business metrics and make data-driven decisions.

The dashboard offers insights into:

- Booking trends over time
- Revenue performance
- Room night utilization
- Cancellation analysis
- Customer loyalty segmentation
- Booking channel effectiveness
- Guest stay duration patterns

---

## 🎯 Business Objectives

- Monitor overall booking performance
- Identify peak booking periods
- Analyze cancellation behavior
- Evaluate loyalty program effectiveness
- Understand customer booking habits
- Track revenue generation across periods
- Optimize channel performance

## 📈 Dashboard Features

### Booking Trend Analysis
- Tracks booking volume over time
- Identifies growth and decline periods

### Day-wise Booking Distribution
- Analyzes bookings by day of the week
- Identifies the highest booking days

### Loyalty Tier Performance
- Compares booking contributions from:
  - Non-member
  - Select
  - Preferred
  - Essential
  - Premier
  - Elite
  - Iconic

### Advance Booking Analysis
- Shows how far in advance guests make reservations
- Helps forecast demand patterns

### Length of Stay Analysis
- Evaluates average guest stay duration
- Identifies common stay lengths

### Booking Channel Analysis
- Measures performance across:
  - Website
  - Mobile App
  - GDS
  - Phone
  - OTA
  - Direct Booking
  - Velora.com

## 📌 Insights Generated

### Customer Behavior
- Majority of bookings come from non-members.
- Guests tend to book one week before their stay.

### Revenue Performance
- Strong revenue generation despite a relatively high cancellation rate.

### Booking Trends
- Mid-week days generate the highest booking volume.

### Loyalty Program
- Significant opportunity exists to convert non-members into loyalty members.

### Channel Performance
- Direct booking channels contribute a substantial share of reservations.
